............DESCRIPCION......................

Implementación del juego clásico Snake usando React (vía CDN) y Babel (vía CDN), todo en un solo archivo index.html sin herramientas de build.
El proyecto está estructurado en componentes reutilizables: App, Game, Board, Snake, Food, Score, StartScreen y GameOverScreen, usando useState y useEffect para manejar el estado y el loop del juego.

...........INSTRUCCIONES...............


Presiona Jugar en la pantalla de inicio para comenzar
Mueve la serpiente con las flechas del teclado:

↑ — arriba
↓ — abajo
← — izquierda
→ — derecha


Come los puntos azules para crecer y sumar puntos
Evita chocar con las paredes o con tu propio cuerpo
El juego termina si hay una colisión — presiona Reiniciar para volver a jugar
Recuerda que la velocidad de la serpiente aumenta automáticamente cada 5 puntos.
